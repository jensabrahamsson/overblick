◈ Överblick

Eight minds.
One framework.
Zero trust.

An open-source, security-first framework for running multiple AI agent personalities — locally, privately, without API keys.

$  python -m overblick run  anomal
⬡
Personalities
8 YAML-driven identities — each with voice, backstory, and psychology
⊛
Security
Fail-closed 6-stage SafeLLM pipeline. No external content reaches the LLM unguarded.
◑
Plugins
15 plugins — Moltbook, IRC, email triage, AI digest, shadow-self generation & more
◇
Local-first
Runs on Qwen3:8b via Ollama. No API keys. Your data stays on your hardware.
▷
Tested
2250+ tests. Personality tests run against a real LLM via the Gateway.
